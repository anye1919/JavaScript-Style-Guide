JavaScript 编码规范
===================

##JavaScript 文件引用

为了加快页面载入速度，尽量避免在 HTML 文件中直接编写 JavaScript 代码，因为这样会大大增加 HTML 文件的大小，
应在 `body` 标签最底下引入 `<script src="filename.js"></script>` 标签，注意：`script` 标签的 `type` 属性是非必要的。
